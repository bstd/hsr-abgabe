# Projektabgaben

## Projekt 1: notez-app -Notizenverwaltung für CAS-FEE HSR 2015

### Requirements
- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/)

### install
clone or download then

    $ cd [path]\notez-app
    $ npm i

### run
    $ cd [path]\notez-app\bin
    $ node www

browse to localhost:3000

### optional: develop build
with grunt
- scripts from source/js are concatenated and minified
- less and css from source/css are compiled, concatenated and minified
- these files are watched and changes trigger build

use default task via cmd:

    $ cd [path]\notez-app
    $ grunt
